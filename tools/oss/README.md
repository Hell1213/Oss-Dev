# OSS Tools

This directory contains tools specific to OSS development workflows.

## Planned Tools

### Git Tools (`git_*.py`)
- `git_status.py` - Check git status
- `git_branch.py` - Create/switch/delete branches
- `git_commit.py` - Create commits
- `git_push.py` - Push branches
- `git_rebase.py` - Rebase operations
- `git_merge.py` - Merge operations
- `git_fetch.py` - Fetch from remotes
- `git_diff.py` - Show diffs

### GitHub API Tools (`github_*.py`)
- `fetch_issue.py` - Get issue details
- `create_pr.py` - Create pull request
- `get_pr_status.py` - Check PR status
- `list_issues.py` - List repository issues
- `check_pr_comments.py` - Get PR comments

### Repository Analysis Tools (`repository_*.py`)
- `analyze_repository.py` - Deep codebase analysis
- `check_start_here.py` - Check for START_HERE.md
- `create_start_here.py` - Generate START_HERE.md
- `update_start_here.py` - Update START_HERE.md
